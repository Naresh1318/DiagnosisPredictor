The Diagnosis Predictor is an application designed to take abnormal lab tests, prescriptions of patients based on the visits to the hospital the previous year and predict the Diagnosis and also give out precautionary measures. The inputs given are in the form of a string which should be a sequnce of abnormal lab tests and prescriptions as shown in the figure below. The result is a table which gives the ICD-9(International Classification of diseases) code of the predicted Diagnosis and the Probability of the patient having that particular diagnosis. A set of steps to reduce the deleterious effect of the diagnosis is displayed as well. This application predicts the ICD-9 code for the patient which has to be done manually in hospitals.
